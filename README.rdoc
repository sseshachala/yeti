== Yeti

Experimental website uptime and performance monitoring SAAS. Using Amazon Payments for metered billing.

==Installation
Yeti can be installed with the chef cookbook here: https://github.com/obi-a/cook

==Technology Stack
- Ruby on Rails
- CouchDB 
- Ragios Server
- Amazon Simple Mail Service
- Amazon Payments
- Ubuntu 

==Built and Tested
Ubuntu 12.10
